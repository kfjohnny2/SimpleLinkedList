# Introdução #
    Código para demonstração de uma lista encadeada simples, representada apenas por números inteiros como conteúdo.


# Compilação #
    Com o terminal dentro da pasta do projeto, execute o comando:

        make

# Composição #
    O projeto possui um Nó principal "SLLNode", que é um struct simples, e é composto por um valor inteiro "miData", que armazena o value do nó;
    Um ponteiro para um nó "mpNext", que irá apontar para o próximo nó.

# Operações #

    As operações do projeto são:
        1- print: Imprime todos os valores dentro do struct SLLNode;
        2- length: Retorna o tamanho da lista;
        3- empty: Verifica se a lista está vazia;
        4- clear: Limpa a lista;
        5- front: Trás o valor do nó mais a frente da lista;
        6- back: Trás o valor do nó mais atrás da lista;
        7- pushFront: adiciona um valor a frente da lista;
        8- pushBack: adiciona um novo valor atrás da lista;
        9- popFront: retira um valor da frente da lista e retorna o mesmo;
        10- popBack: retira um valor de trás da lista e retorna o mesmo;
        11- find: encontra um valor solicitado na lista;
        12- insert: insere um valor em uma determinada posição da lista;
        13- remove: remove um determinado valor da lista;
